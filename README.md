# Simple Interest Calculator

A simple, lightweight application designed to calculate the simple interest accrued on a principal amount over a specified period at a given interest rate.

## Formula Used
The application calculates interest using the standard mathematical formula:

$$A = P(1 + rt)$$

Where:
* **$A$** = Total Accumulated Amount (Principal + Interest)
* **$P$** = Principal Amount
* **$r$** = Annual Interest Rate (as a decimal)
* **$t$** = Time (in years)

## Features
* Input validation for numbers, percentages, and timeframes.
* Instant calculation output.
* Responsive and user-friendly interface.
